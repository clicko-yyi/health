1. 维生素分为水溶性维生素和脂溶性维生素

2. 水溶性维生素包括：B族维生素和维生素C

3. 脂溶性维生素包括：维生素A、D、E和K

4. 水溶性维生素的特点：溶与水、在肠道吸收进入血液、不在体内存储

5. 脂溶性维生素的特点：溶与脂肪、在肠道吸收时随脂肪经淋巴系统吸收、摄入后大部分存在与脂肪组织

6. 维生素A，也称为视黄醇

   1. 生理功能：视觉功能、维持皮肤粘膜完整性、维生素A原类胡萝卜素是重要的抗氧化物质
   2. 缺乏症：夜盲症、干眼症、皮肤粘膜的改变
   3. 来源：植物来源（维生素A原类五萝卜素），动物来源（预先形成的维生素A，视黄醇）
   4. 过量：肝脏损伤、骨质疏松
   5. 推荐摄入量：800/ugRAE/d/男，700/ugRAE/d/女

7. 维生素D，可以体内自动合成、晒太阳就可以

   1. 生理功能：促进钙的吸收、发挥激素作用
   2. 缺乏症：佝偻病、骨软化症、估值疏松
   3. 过量：中毒、肾结石、骨组织钙化
   4. 推荐摄入量：10ug/d
   5. 来源：鱼肝油、奶酪和蛋黄

8. 维生素E，称为生育酚，重要的脂溶性抗氧化剂

   1. 生理功能：抗氧化的作用、改善生育功能、维持免疫功能、维持红细胞的完整性
   2. 缺乏：自然界广泛存在，不会缺乏、多出现在脂肪吸收不良的患者
   3. 过量：中毒（毒性较弱）
   4. 来源：油脂丰富的种子（花生、芝麻）、大豆、谷胚

9. 维生素K，有K1,K2,K3

   1. 生理功能：参与凝血、参与骨代谢、与心血管健康有关
   2. 缺乏：凝血时间长
   3. 过量：研究较少
   4. 来源：菜籽油、黄瓜、大豆和菠菜等阿斯顿撒

10. 维生素B1，硫胺素、抗脚气病因子、对碱性环境极不稳定、易被氧化失活、在空肠和回肠内吸收、从尿中排出

    1. 生理功能：构成辅酶、促进胃肠道蠕动、维持正常的神经组织功能
    2. 缺乏：脚气病
    3. 推荐摄入量：1.4/md/d/man  14/md/d/women
    4. 来源：未精制的谷类、瘦肉及动物内脏、坚果，豆类和薯类。

11. 维生素B2，核黄素

    1. 生理功能：参与氨基酸、脂肪和碳水化合物代谢、参与骨髓中红细胞生成以及铁的吸收
    2. 缺乏：脂溢性皮炎、免疫功能低下
    3. 推荐摄入量：1.4/md/d/man, 1.2/md/women
    4. 来源：肉类、内脏、奶类、蛋类和坚果

12. 烟酸，性质稳定，可以用色氨酸合成，氨基化合物烟酰胺

    1. 生理功能：减低血脂、作为某些酶的辅酶，参与氧化还原反应、维持葡萄糖耐量，减低血脂
    2. 缺乏：赖皮病，3D症状（皮炎、腹泻和痴呆）
    3. 推荐摄入量：15/mgNE/d/man, 12/mgNE/women
    4. 来源：肝、肾、瘦肉、鱼和坚果，谷物的烟酸80-90存在于种皮中，精致后含量低

13. 维生素B6，吡多醇

    1. 生理功能：参与氨基酸代谢

14. 维生素B12，钴胺素，含有金属的维生素、分子量最大的维生素

    1. 生理功能：作为辅酶因子参与蛋氨酸的体内合成
    2. 来源：肉类、动物内脏
    3. 推荐摄入量：2.4ug/d

15. 叶酸，烹调过程中损失率高，最先在菠菜中发现

    1. 生理功能：略
    2. 缺乏：巨幼红细胞贫血
    3. 来源：猪肝、黄豆
    4. 推荐摄入量：400/ugDFE/d

16. 维生素C，抗坏血酸，抗氧化、自身不能合成，酸性环境稳定

    1. 生理功能：抗氧化作用、协助非血红素铁的吸收
    2. 缺乏：草酸结石、铁吸收过度
    3. 来源：酸枣、鲜枣、红辣椒
    4. 推荐摄入量：100/mg/d

    11. 维生素的毒副作用主要取决于视黄醇及视黄酰酯的摄入量，并与机体的生理及营养状况有关。急性毒性产生于一次或多次连续摄入大量的维生素A（成人大于推荐摄入量的100倍左右，儿童大于推荐摄入量的20倍左右），维生素A过量的临床表现包括恶心、呕吐、头痛、脑脊液压力升高、眩晕、视力模糊、肌肉不协调、严重皮疹。维生素A过量可致胚胎畸形、增加骨矿物质丢失和骨质疏松症风险以及引起肝脏损伤，故本题选D
    12. 维生素A又叫抗干眼症维生素、视黄醇。维生素A的缺失，最早表现为暗适应能力下降，严重者可致夜盲症，还可引起干眼病，进一步发展可导致失明。维生素B1又叫硫胺素，硫胺素的缺乏又称脚气病，主要损害神经—血管系统。烟酸又称尼克酸、维生素B5、维生素PP、抗癞皮病因子等。烟酸缺乏时即出现癞皮病，其典型症状是皮炎（dermatitis）、腹泻（diarrhoea）、痴呆（dementia），即3D症状。维生素C的缺乏容易导致坏血病的发生，故本题选A。

    32. 怀孕早期缺乏叶酸是引起胎儿神经管畸形（NTD）的主要原因。NTD是指在胚胎发育的3～4周神经管未闭合造成的先天缺陷，主要表现为脊柱裂和无脑畸形等中枢神经系统发育异常。叶酸摄入不足还易导致孕妇流产率增高、胎儿宫内发育迟缓、早产及新生儿低出生体重等。因此，妇女在孕前至孕早期及时补充叶酸，可有效地预防70%以上神经管缺陷的发生。骨骼畸形是由于缺乏维生素D与钙所造成的
